This is a project for predicting LendingClub loan charge-offs. 
The data is available on kaggle website: https://www.kaggle.com/wordsforthewise/lending-club. Unzipping the data to the data folder.
1. Project_Defination.sas:  create folders and global macro variables.
2. DataPrep.sas: process ETL.
3. AppData: create training and validation dataset
4. RiskModel.sas: predicting modeling and model stability checking.
The macros are stored in cmacros folder.
